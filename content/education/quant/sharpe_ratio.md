# Sharpe Ratio

## Beginner Level

### What Is It?
The Sharpe ratio measures risk-adjusted return by dividing excess return by standard deviation.

### Origin
William Sharpe introduced the ratio in 1966.

### Why It Matters
The Sharpe ratio is the standard metric for comparing strategy efficiency.

## Intermediate Level

### Market Mechanics
Sharpe uses annualized excess return over the risk-free rate divided by annualized volatility.

### How It Behaves
Sharpe above 1.0 is considered good; values above 2.0 are exceptional.

### Key Data to Watch
- Annualized Sharpe
- Rolling Sharpe
- Drawdown-adjusted variants

## Advanced Level

### Institutional Behavior
Institutions target minimum Sharpe thresholds for strategy approval.

### Professional Use Cases
- Strategy ranking and capital allocation
- Performance attribution

### AI Interpretation in Systems Like Arkhe
- **Portfolio Agent**: Optimizes for portfolio-level Sharpe.

### Key Takeaways
The Sharpe ratio quantifies return per unit of risk.
